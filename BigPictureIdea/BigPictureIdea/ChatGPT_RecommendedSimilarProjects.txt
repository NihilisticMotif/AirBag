4. GraphHopper
    * Description  
        * It is an open-source route planning and navigation tool that can handle large-scale routing tasks.
    * Purpose 
        1. It help provide travel times, distances, turn instructions and path (route) between 2 locations.
          It also takes travel time variations of peak hours and off-hours into account. 
        2. It can solve Travelling Salesman Problem (deliver items or services to your customers).
        3. It can tell the name of the selected or your current location and vice versa (Geocoding).
    * Note
        1. This software have free version and paid version.
        2. GraphHopper Directions API is RESTful web services. It suppose C#, Ruby, PHP, Python, JavaScript, Postman etc.
        * https://www.postman.com/api-evangelist/graphhopper/collection/p083tjy/graphhopper-directions-api
    * Keyword
        1. Travelling Salesman Problem
        * This problem asks the following question: 
          "Given a list of cities and the distances between each pair of cities, 
          what is the shortest possible route that visits each city exactly once and returns to the origin city?"
          It is an NP-hard problem. (https://en.wikipedia.org/wiki/Travelling_salesman_problem)
        2. Map Tiles 
        * It is the image file that represents a specific geographic area at a particular zoom level.
        3. Geocoding 
        * It is the process of transforming a description of a location—such as a pair of coordinates, 
        an address, or a name of a place—to a location on the earth's surface.
        (https://desktop.arcgis.com/en/arcmap/latest/manage-data/geocoding/what-is-geocoding.htm)
    * URL:
