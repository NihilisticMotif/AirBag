How can I get involved?
    1. Anyone can help by improving map data!
    2. Technically, OpenStreetMap is a database, and not what everyday language calls a 'map'. There are many software applications, known as editors, that can be used to edit the database. Before you can edit any data you will need to create an OSM account.
